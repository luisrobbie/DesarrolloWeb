# Desarrollo Web

Repositorio que contiene proyectos realizados en la materia
de Desarrollo Web, incluyendo aplicaciones con backend y base de datos.

## Proyecto: Reconstruye
Aplicación web con arquitectura cliente-servidor, desarrollada
con frontend, backend en PHP y base de datos MySQL.

### Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Procedimientos almacenados

### Características
- Interfaz web dinámica
- Conexión a base de datos MySQL
- Uso de procedimientos almacenados para la lógica de datos
- Aplicación levantada en entorno local y expuesta para acceso remoto
